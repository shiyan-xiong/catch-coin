# 🪙 接金币游戏 (CatchCoin)

一个Vibe Coding的接金币网页游戏，使用 HTML + CSS + JavaScript 实现。

## 🎮 游戏介绍

在游戏中，你需要控制钱袋子接住从天而降的金币，同时避开石头。大金币价值更高，但石头会扣分！

## ✨ 游戏特色

- 🎨 **现代化设计**：采用现代化 UI 设计，视觉效果流畅
- 🎯 **三种物品**：
  - 🪙 大金币：+2分
  - 🪙 小金币：+1分
  - 🪨 石头：-1分
- ⚡ **动态速度**：游戏进行中速度会逐渐增加，挑战性不断提升
- 🏆 **最高分记录**：自动保存最高分到本地存储
- ⚙️ **个性化设置**：可自定义背景颜色和主题
- 🎮 **控制方式**：支持鼠标控制

## 🎯 游戏规则

1. 点击"开始"按钮开始游戏
2. 使用鼠标移动控制钱袋子左右移动
3. 接住金币得分，避开石头
4. 游戏速度会随时间逐渐增加
5. 漏接50个金币游戏结束
6. 点击"暂停"可以暂停游戏
7. 点击"重玩"可以重新开始游戏

## 🕹️ 操作方式

### 鼠标控制
- 在游戏区域内移动鼠标，钱袋子会跟随鼠标位置

## 🎨 界面说明

### 游戏信息栏
- **得分**：当前游戏得分
- **漏接**：漏接的金币数量，达到50个游戏结束
- **最佳**：历史最高得分

### 控制按钮
- **开始/暂停**：开始新游戏或暂停/继续已开始的游戏
- **重玩**：重新开始游戏

### 设置面板
点击右上角的 ⚙️ 图标打开设置面板，可以：
- 选择预设主题（草原、薰衣草、黑夜、白日梦）
- 自定义网页背景颜色
- 自定义游戏区域背景颜色

## 🎮 游戏技巧

1. **保持移动**：钱袋子移动速度有限，提前预判物品掉落位置
2. **优先接大金币**：大金币得分更高，但也要注意避开石头
3. **速度管理**：随着速度增加，反应要更快
4. **合理暂停**：需要休息时可以暂停游戏
5. **注意漏接数量**：漏接50个金币游戏就会结束
